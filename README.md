### 用法

- Android端sfa: 用 https://sing-box-subscribe-sable-gamma.vercel.app/config/你的订阅链接&file=1 当做订阅地址使用
- Win端：在浏览器中访问 https://sing-box-subscribe-sable-gamma.vercel.app/config/你的订阅链接&file=4 ，保存内容即可在singbox中使用
